# PITEZA premium multipage website

Многостраничный статический сайт питомника PITEZA.

## Структура
- index.html — главная страница
- puppies.html — полный каталог щенков
- puppy-single.html — карточка щенка
- breed-maltipoo.html — страница породы
- about.html — о питомнике и родителях
- how-to-buy.html — процесс покупки
- stories.html — отзывы и истории семей
- contact.html — контакты и форма заявки
- assets/css/styles.css — общий визуальный стиль
- assets/js/site.js — меню, галерея, FAQ, анимации, поиск по карточкам
- assets/images — изображения

Сайт открывается локально через index.html и готов к загрузке на статический хостинг. Для реальной отправки формы нужно подключить endpoint/CRM/Telegram-бота.
